# plaoc-docs

plaoc 前后端开发文档,构建符合开发者直觉的 API。

文档站点：[地址（国外）](https://bioforestchain.github.io/plaoc-docs/)，[国内](http://docs.plaoc.com)

## 快速启动

```bash
yarn

yarn dev
```

## 生成静态文件

```bash
yarn build
```
