# plaoc-docs

plaoc 前后端开发文档,构建符合开发者直觉的 API。

文档站点：[地址](https://bioforestchain.github.io/plaoc-docs/)

> 有写文档的都是测试过的，大哥们多多看看 API 合不合理，尽量向 element-ui 的风格靠拢，让开发者无缝衔接.

## 快速启动

```bash
yarn

yarn dev
```

## 生成静态文件

```bash
yarn build
```
