# plaoc-docs

plaoc 前后端开发文档,构建符合开发者直觉的 API。

文档站点：[docs.plaoc.com](https://docs.plaoc.com)

DAPP商城地址：[shop.plaoc.com](https://shop.plaoc.com)

## 快速启动

```bash
yarn

yarn dev
```

## 生成静态文件

```bash
yarn build
```
