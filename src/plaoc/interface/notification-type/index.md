用来做 [NotificationOptions](../notification-options/index.md)接口中type属性的值；
NotificationType是一个string;
合法的值如下：

- "SUCCESS"

  任务成功通知类型


- "WARNING"

  任务产生警告的通知类型


- "ERROR"

  任务失败的通知类型

| "WARNING"     | ✅      | ✅  | ✅          | X       |
