Restart the front and back ends of the application;

```js
DwebServiceWorkerPlugin.restart()
```

## parameters

  - none

## return value

  Return a Promise, resolve to fulfill a boolean;

## Example
```js
import  { dwebServiceWorkerPlugin } from "@plaoc/plugin";
dwebServiceWorkerPlugin.restart()
```
