获取app更新控制器

```js
DwebServiceWorkerPlugin.update()
```

## parameters

  - none

## return value

  返回一个 [UpdateControllerPlugin](../update-controller/index.md);  

## Example
```js
import { DwebServiceWorkerPlugin } from "@plaoc/plugin";
const updateControllerPlugin = DwebServiceWorkerPlugin.update();
```
