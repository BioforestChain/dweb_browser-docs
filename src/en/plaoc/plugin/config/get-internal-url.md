Get the internal Url;

```js
configPlugin.getInternalUrl()
```

## parameters

  - none

## return value
  
  Return a Url of an internal address;

## Example
```js
import { configPlugin } from "@plaoc/plugin";
configPlugin.getInternalUrl()
```
