Set the public Url;

```js
configPlugin.setPublicUrl(url)
```

## parameters

  - url

    The parameter is the new url;
    The type of the parameter is string;


## return value
  
  Return a Url of a public address;

## Example
```js
import { configPlugin } from "@plaoc/plugin";
configPlugin.setPublicUrl("http://xxx")
```
