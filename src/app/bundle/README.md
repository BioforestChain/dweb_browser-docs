DwebBrowser前后端打包工具，把您开发的包打包成`.jmm`形式。



### @bfex/dweb-jmm-bundler

```
 全局安装指令
 # npm方式
   npm i @bfex/dweb-jmm-bundler -g

 # yarn方式
   yarn global add @bfex/dweb-jmm-bundler
```

#### 使用

> jmm
