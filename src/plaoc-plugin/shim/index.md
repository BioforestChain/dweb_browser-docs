# Plaoc Shim

一个在javascript范围内提供跨平台的访问系统功能的垫片；

## 安装
```bash
npm install @plaoc/plugins
```

## 示例

```js
import { dwebServiceWorker } from "@plaoc/plugins";
dwebServiceWorker.addEventListener('pause', () => {
  // ......
})
```
