隐藏系统导航栏

```js
NavigationBarPlugin.hide()
```

## 参数

  - 无

## 返回值

  - 一个Promise对象， resolve兑现null

## 示例
```js
import { NavigationBarPlugin } from "@plaoc/plugins";
const value = NavigationBarPlugin.hide()
```
