获取内部的Url;

```js
configPlugin.getInternalUrl()
```

## 参数

  - 无

## 返回值
  
  返回一个内部地址的Url;

## 示例
```js
import { configPlugin } from "@plaoc/plugins";
configPlugin.getInternalUrl()
```
