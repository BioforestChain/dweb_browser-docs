NotificationOptions是[HapticsPlugin.notification()](../../plugin/haptics/notification.md)函数的参数类型;


## 属性

  - type

    属性用来设定生成通知返回的风格；
    属性值的类型是[NotificationType](../notification-type/index.md);
    默认值是字符串 "SUCCESS";
