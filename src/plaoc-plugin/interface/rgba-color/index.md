AgbaColor 接口是一个用来描述颜色信息的对象；
 

## 属性

- AgbaColor.red
  
  值是一个0~255之间的数字，用来描述红色；

- AgbaColor.green
  
  值是一个0~255之间的数字，用来描述绿色；

- AgbaColor.blue
  
  值是一个0~255之间的数字，用来描述蓝色；

- AgbaColor.alpha
  
  值是一个0~255之间的数字，用来描述不透明度；
