[ToastShowOptions.duration](../toast-show-options/index.md)属性值的类型接口;
指的类型是一个 string; 合法的值范围如下；

- "long"

  长时间的显示；约 3500ms


- "short"
 
  短时间的显示；约 2000ms;
 

## 平台兼容性

| 属性/方法/事件 | Android | IOS | Desktop-Dev | Desktop |
|:------------:|:-------:|:---:|:-----------:|:-------:|
| long         | ✅      | ✅  | ✅           | X       |
| short        | ✅      | ✅  | ✅           | X       |
